Yeoman generator with:

- Typescript
- Prettier
- Jest
- Convetional Commits with standard-version
- Husky git hooks
- React (optional)

## Usage

```sh
npm i -g yo @leddgroup/generator-typescript-project

mkdir my-new-ts-project && cd my-new-ts-project
yo typescript-project
```
